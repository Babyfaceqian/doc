1. 使用img标签时

    为img标签添加onerror方法，当src指向的图片路径加载失败后，调用onerror方法手动修改this.src为默认图片

    代码：
    ```html
    <img src="xxxx.jpg" onerror="nofind()" />
      function nofind(){
      var img=
      event.srcElement;
      img.src="default.jpg
      img.onerror=null;//避免抖动
    }
    ```
